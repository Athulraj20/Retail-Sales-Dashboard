# Retail-Sales-Dashboard


Project Overview

This project analyzes retail sales transaction data using Python (Pandas) and Power BI.

The objective was to clean raw sales data, standardize product information, create meaningful product categories, and build an interactive dashboard to generate business insights and support decision-making.

---

Tools Used

- Python
- Pandas
- Jupyter Notebook
- Power BI
- GitHub

---

Dataset Information

The dataset contains:

- 166,000+ retail sales transactions
- Product information
- Customer types
- Order details
- Pricing and discount data
- Quantity sold

---

Data Cleaning & Transformation

The following data preparation steps were performed:

- Removed and corrected inconsistent records
- Standardized product names
- Created a Product_Standard column
- Created Product_Category classifications
- Extracted pack size information
- Calculated Product Sales values
- Checked and handled missing values
- Standardized month names

Key Insights

- Rice generated the highest category revenue.
- Banana products showed strong customer demand.
- Retail Price customers contributed the largest share of revenue.
- Coconut and Edible Oil categories performed consistently well.
- A small group of products contributed a significant share of overall sales.
